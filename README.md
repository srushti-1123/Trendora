# Trendora - Ecommerce Web Application

## Technologies Used
- HTML
- CSS
- JavaScript
- Java
- JDBC
- Servlets
- Spring Boot
- MySQL
- Maven

## Features
- User Login & Registration
- Product Listing
- Cart Management
- Order Placement
- Admin Dashboard

## Tools Used
- Eclipse IDE
- Apache Tomcat
- MySQL Workbench

## How to Run
1. Import project into Eclipse
2. Configure MySQL database
3. Run on Apache Tomcat Server
